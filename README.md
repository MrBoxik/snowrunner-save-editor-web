# SnowRunner Save Editor (Web)

Static browser app version for GitHub Pages.

## Runtime files

- `index.html`
- `styles.css`
- `app.js`
- `app_icon.ico`
- `assets/`
- `vendor/`

## Local run

Open `index.html` in a browser.

## Deploy

See `INSTRUCTIONS.txt` for full release steps:
- new repository deployment
- adding this web app to an existing repository

## Notes

- Objectives+ catalog is already embedded in `app.js`.
- Edits run fully in-browser; no file upload to any server by the app itself.
